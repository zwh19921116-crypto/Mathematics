# Number Line Explorer

Interactive, beginner-friendly number line app for teaching:

- Addition
- Subtraction
- Multiplication (repeated jumps)
- Division (equal jumps)

## Run Locally

Open `index.html` in a browser.

## Deploy To GitHub Pages

1. Push these files to your repository root.
2. In GitHub, open **Settings** -> **Pages**.
3. Under **Build and deployment**, set:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main` (or your default branch)
   - **Folder**: `/ (root)`
4. Save. GitHub will publish your site.

Your site URL will usually be:

`https://zwh19921116-crypto.github.io/Mathematics-NumberLineExplorer/`

## Teaching Notes

- Addition: start at the first number, then jump right/left by the second number.
- Subtraction: start at the first number, then move in the opposite direction of the second number.
- Multiplication: repeated equal jumps.
- Division: count equal jumps to reach the dividend.

For division visualization clarity, the app currently requires exact integer division.
