# Prime Number Explorer

A student-friendly web app to learn and explore prime numbers.

## What this app does

- Explains what prime numbers are in simple language.
- Lets students check if one number is prime.
- Shows a full list of prime numbers in any chosen range.

## Run locally

1. Open the project folder.
2. Double-click `index.html`.

Or run a local server (recommended):

```powershell
cd "E:\Edgeducate\Software\Mathematics\Prime Number Explorer"
python -m http.server 8000
```

Then open <http://localhost:8000>.

## Teaching ideas

- Ask students to predict whether a number is prime before checking.
- Explore ranges like 1-50, 50-100, and 100-200.
- Discuss why 1 is not prime and why 2 is the only even prime.
