# Fraction Visualiser

A classroom-friendly web app to help students understand how fractions work.

## Features

- Visualise a single fraction with pie charts.
- See simplified fraction, decimal form, and mixed number form.
- Learn fraction arithmetic with worked steps:
  - Addition
  - Subtraction
  - Multiplication
  - Division (using reciprocal)
- Compare fraction A, fraction B, and the result using pie charts.
- Responsive layout for desktop and mobile.

## How to Run

1. Open the project folder.
2. Open `index.html` in a web browser.

No installation is required.

## Classroom Ideas

- Ask students to predict the result before clicking **Show Steps**.
- Use improper fractions (for example `9/4`) to show multiple whole pies plus a remainder.
- Compare operations and discuss why only add/subtract require common denominators.
